# AI vs Human Text Detector

A fine-tuned DistilBERT model that classifies whether a given text was written by a human or generated by AI. Built as a minor project with a Flask web interface and SHAP-based explainability.

---

## Project Structure

```
├── config.py              # All settings (paths, hyperparameters, thresholds)
├── train_distilbert.py    # Training script — run once
├── predict.py             # Terminal-based inference
├── app.py                 # Flask web interface
├── requirements.txt       # Python dependencies
└── README.md
```

---

## Dataset

We used the **AI vs Human Text** dataset from Kaggle:
- 100,000 samples (50k human, 50k AI-generated)
- Columns: `text`, `generated` (0 = human, 1 = AI)
- Download from: https://www.kaggle.com/datasets/shanegerami/ai-vs-human-text

Place the CSV at the path set in `config.py`:
```python
CSV_PATH = r"C:\Users\rival\Downloads\balanced_ai_human_100k.csv"
```

## Setup

1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/ai-vs-human-detector.git
cd ai-vs-human-detector
```

2. Create a virtual environment
```bash
python -m venv .venv
.venv\Scripts\activate        # Windows
source .venv/bin/activate     # Mac/Linux
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

4. Download the dataset from Kaggle and update `CSV_PATH` in `config.py`

---

## Usage

### Step 1 — Train (run once)
```bash
python train_distilbert.py
```
Model is saved to `./distilbert_ai_vs_human/`

### Step 2 — Predict (terminal)
```bash
python predict.py
```
Paste text, type `END` on a new line to submit. Type `explain` to see SHAP word importance.

### Step 3 — Web Interface
```bash
python app.py
```
Open http://localhost:5000 in your browser.

## Features

- Fine-tuned DistilBERT with custom dropout and class-weighted loss
- Early stopping to prevent overfitting
- Live training graphs (matplotlib)
- Long text support — automatically chunks and averages predictions
- SHAP explainability — shows which words pushed the prediction toward AI or Human
- Flask web interface with prediction confidence bars

---

## Known Limitations

- Formal or polished human writing (Wikipedia, academic papers) may be misclassified as AI due to dataset bias
- The training dataset contains mostly student-style human writing, so the model performs best on that style
- This limitation is consistent with published findings on commercial AI detectors (GPTZero, Turnitin)
